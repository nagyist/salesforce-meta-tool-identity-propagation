# Video Script — Text Only (No Voiceover)

**Duration:** 60–75 seconds
**Format:** On-screen text captions + screen recording + animated diagram
**Music:** Subtle lo-fi or tech ambient (royalty-free)
**Style:** Bold text appears/fades on screen — Greg Isenberg style

---

## SCENE 1 — Hook (0:00–0:05)
**[Black background, bold white text fades in]**

> What if your AI agent in Teams
> could query Salesforce... as YOU?

**[Screen recording: Microsoft Teams]**
**[Text overlay at top:]**

**[Show typing in Teams chat:]**

> "Give me my open opportunities"

## SCENE 3 — Behind the Scenes (0:12–0:15)
**[Black background, text fades in]**

## SCENE 4 — Diagram Animation (0:15–0:45)
**[Excalidraw diagram animates step by step]**

**[0:15 — Boxes appear]**

> Teams → AI Foundry → APIM → Salesforce

**[0:19 — Question bubble appears]**
Text overlay:

> The user's Azure AD token travels with the request.

**[0:24 — Forward arrows animate]**
Text overlay:

> Each hop carries the user's identity.

**[0:29 — Token Exchange box appears]**
Text overlay:

> APIM validates the token.
> Maps Azure AD identity → Salesforce username.
> Exchanges for a per-user SF access token.

**[0:37 — SF Access Token arrow appears]**
Text overlay:

> No shared service accounts. No credential leaks.

**[0:41 — Return arrows animate]**
Text overlay:

> Salesforce enforces the user's own permissions.
> Data flows back through the same chain.

## Editing Tips

- **Font:** Use a clean sans-serif (Inter, Helvetica, or SF Pro)
- **Text animation:** Simple fade-in or slide-up. No flashy transitions.
- **Text size:** Large enough to read on mobile (LinkedIn = mostly mobile)
- **Music:** Try "Chillhop" or "LoFi Study" playlists on Pixabay (royalty-free)
- **Format:** 1080x1350 (portrait) for LinkedIn mobile
- **Tool:** CapCut (free) handles text overlays + timing perfectly
- **Timing:** Let each text stay on screen ~3 seconds minimum for readability
